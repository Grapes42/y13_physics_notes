## Equations
- $V = Ed$
- $\Delta E = Vq$
- $E = \frac{1}{2}QV$
- $Q = CV$
- $C = \frac{\xi_o \xi_r A}{d}$
- $C_T = C_1 + C_2 + ...$
- $\frac{1}{C_t}=\frac{1}{C_1}+\frac{1}{C_2}+...$
- $\tau = RC$
- $R_t = R_1 + R_2 + ...$
- $\frac{1}{R_t} = \frac{1}{R_1} + \frac{1}{R_2} + ...$
- $V = IR$
- $P = VI$
- $\phi = BA$
- $\xi = -L \frac{\Delta I}{\Delta t}$
- $\xi = - \frac{\Delta \phi}{\Delta t}$
- $f_o = \frac{1}{2 \pi \sqrt{LC}}$
- $\frac{N_p}{N_s} = \frac{V_p}{V_s}$
